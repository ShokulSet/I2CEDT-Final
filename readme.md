# Real Estate Sale Helper

Web application สำหรับการช่วย Sale หาอสังหาให้ลูกค้าด้วย LLM กับฐานข้อมูลขนาดใหญ่

## Setup frontend command

```bash
cd frontend
npm i
npm start
```

## Setup backend command

```bash
cd backend
npm i
npm start
```
